# Superstore Sales Data Analytics Project

## Project Overview
This project performs a complete data analytics process on a retail Superstore dataset. The objective is to analyze sales performance, profitability, discount impact, and regional trends to generate actionable business insights.

## Business Problem
The retail company wants to:
- Identify high-performing regions
- Analyze profit by product category
- Understand the impact of discounts on profit
- Examine seasonal sales patterns

## Dataset
- Source: Kaggle Superstore Sales Dataset
- Records: Sales transactions
- Key Variables: Sales, Profit, Discount, Quantity, Category, Region, Order Date

## Tools & Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

## Analysis Performed
- Data cleaning & preprocessing
- Exploratory Data Analysis (EDA)
- Sales distribution analysis
- Regional performance analysis
- Profitability analysis
- Correlation analysis
- Linear regression (Discount vs Profit)
- Time-series analysis (Monthly Sales Trend)
